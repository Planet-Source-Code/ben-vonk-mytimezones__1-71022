Notice for Add, Delete or Modify an favorit clock
- Add a favorit clock
  To add a favorit, select the favorit clock you want.
  Then select a timzone in the timezones combobox above the buttons.
  After that you press the 'Add favorit' button to add the
  new favorit zone to the selected clock.

- Delete a favorit clock
  Select the favorit clock you want to delete.
  Then press the 'Delete favorit' button and the timezone
  will be deleted.

- Modify a favorit clock or zone clock
  Double click on the favorit clock you want to modify.
  Then clock the information window wil be displayed.
  In the button bar you can make the changes for it.
  Settings like; picture, clock name, alarm time and message.
  For an zone clock you can only change the picture and clock name.
